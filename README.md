# Smart Student Scoreboard

A modern React-based dashboard to manage student scores with a clean neon UI.

## 🚀 Features
- Add new students
- Update scores
- Automatic PASS / FAIL status
- Average score calculation
- Clean futuristic UI design

## 🛠 Tech Stack
- React (Vite)
- JavaScript
- CSS (Neon UI)

## ▶️ Run Locally
```bash
npm install
npm run dev
